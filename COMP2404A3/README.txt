Compliling:
- Type "make"

Launching:
- Type "./a3"

Usage:
In Admin menu:
- Type "1" to view all students in the school
- Type "2" to view all courses in the term
- Type "3" to view all registrations made for students
- Type "4" to go into a students menu 
- Type "0" to exit

In Student menu:
- Type "1" to view all courses in a term
- Type "2" to view all registrations for that student
- Type "3" to register a course for the student
- Type "0" to exit
